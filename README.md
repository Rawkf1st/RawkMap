# RawkMap

Мой самописный аддон для WoW Sirus — лёгкий, быстрый и максимально простой. 

Делал для своего удобства, чтобы легче было собирать траву, руду и прочее. Задолбался щурить глаза на мизерную миникарту!

### Особенности:
* **Управление:** Масштаб изменяется наведением курсора на миникарту и прокруткой колёсика мыши.
* **Сохранение:** Настройки масштаба сохраняются при перезагрузке интерфейса (/reload) и выходе из игры.
* **Совместимость:** Автоматически сдвигает кастомные баффы и дебафы Сируса влево при увеличении карты, чтобы интерфейс не перекрывался.

### Установка:
1. Скачайте архив RawkMap.zip из раздела Releases.
2. Распакуйте папку RawkMap в директорию World of Warcraft Sirus\Interface\AddOns\.
3. Полностью перезапустите игру.

Тестите, пишите комменты!
